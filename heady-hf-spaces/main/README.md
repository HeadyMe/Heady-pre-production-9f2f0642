# 🐝 Heady AI — Brain Demo

Interactive demo of the Heady ecosystem: a 77-service AI orchestration platform with liquid gateway routing, multi-provider AI inference, and Sacred Geometry branding.

## Features

- **Live AI Chat** — Chat with HeadyBrain (powered by Qwen3-235B via HuggingFace Inference API)
- **Code Editor (IDE)** — Write and run code against Heady AI with Monaco Editor
- **Service Explorer** — Browse all 77 services across 14 groups with visual charts
- **Gateway Dashboard** — See how Heady routes requests across providers
- **Architecture Visualizer** — Interactive system diagram

## Stack

- HTML/CSS/JS (Static SDK)
- Monaco Editor (VS Code core) for the IDE tab
- HuggingFace Inference API for AI chat & code
- Sacred Geometry visual identity
